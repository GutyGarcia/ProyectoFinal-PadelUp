Pasos a seguir para iniciar la aplicación

	1. Crear base de datos en workbench con nombre => padelup
	2. Lanzar el programa desde el Ide
	3. Una vez lanzado, el programa creará la estructura de la bases de datos, sin equipos añadidos
	   debido al número elevado de ellos.
	4. Abrir el archivo scripEquipos y meterlos en en la base de datos desde workbench si se desea

Seguidos estos pasos ya se podrá testear la aplicación con todas sus funcionalidades activas.

**Para que la opción de envio de emails funcione correctamente hay que habilitar la opción "acceso de aplicaciones poco seguras"
  en la cuenta de Google que se vaya a utilizar**
